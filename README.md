# Vehicle Demand Forecasting

## Overview
Exploratory data analysis and demand forecasting applied to 
vehicle rental time series data (Bangalore, 2018-2021).

## Objectives
- Identify demand patterns by hour, day and month
- Build a predictive model to forecast vehicle demand
- Extract actionable insights for fleet and pricing decisions

## Key Findings
- Peak demand occurs between 12h and 17h
- Clear seasonal patterns identified across 2018-2021
- Linear Regression model achieved MAE of 31 cars (mean demand: 74)

## Tools & Libraries
Python | pandas | matplotlib | seaborn | scikit-learn
